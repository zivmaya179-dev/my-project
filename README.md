# 📚 המילים של לני - אפליקציית כרטיסיות לבגרות באנגלית

אפליקציית כרטיסיות (Flashcards) ללימוד מילים לבגרות אנגלית 4 יחידות.

## פתיחת האפליקציה

האפליקציה מתארחת על GitHub Pages. לאחר הפעלת Pages, הקישור יהיה:
`https://<username>.github.io/<repo>/`

מעבירים את הקישור בוואטסאפ ללני - היא נכנסת מהטלפון והכל עובד מיד.

## תכונות

- כרטיסיות מתהפכות - אנגלית מצד אחד, עברית מצד שני
- הקראה אוטומטית של המילה באנגלית (Web Speech API)
- מעקב התקדמות: חדשות / בלימוד / ידועות (נשמר אוטומטית בדפדפן)
- Swipe - החלקה ימינה = יודעת, החלקה שמאלה = עוד לא
- חלוקה ליחידות - אפשר לתרגל יחידה אחת בכל פעם
- ערבוב של הסדר
- מצב מיקוד - לתרגל רק מילים שעדיין לא נשלטו
- מותאם למובייל - עיצוב מודרני, RTL, ללא צורך באינסטלציה

## עריכת רשימת המילים

כל המילים נמצאות בקובץ אחד: [`words.js`](words.js)

פורמט:
```js
{ en: "achieve", he: "להשיג, להגשים", example: "She worked hard to achieve her goals.", unit: "Unit 1" }
```

השדות `example` ו-`unit` אופציונליים. אפשר להעתיק/להדביק את כל הרשימה ולערוך בלי לגעת בקוד.

## קיצורי מקשים (לבדיקה במחשב)

- `Space` - הפוך כרטיס
- `←` - יודעת (במצב RTL)
- `→` - עוד לא
- `↓` - דלג
